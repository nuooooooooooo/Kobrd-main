Get started now

Typing on a Korean keyboard, it's easier than you think.

Have you recently started learning orean but find yourself having to constantly look up certain hangeul placements? You are not alone! In fact, I daresay this is a problem most learners of the Korean language struggle with once they start typing on a computer keyboard — myself included.

But fret not! With Kobrd, you'll start learning Korean dactylography from the beginning, the same way you did when you first learned how to type on a QWERTY keyboard. Icing on the cake: You can learn it all for FREE!

It's free

Yes! You read it right, this application is available for free, because as a learner myself, I believe learning how to type in Korean is an essential step to improve one's mastery in the language. So no more excuses, start learning today!

It's gamified

While learning can be fun, it's sometimes hard to stay motivated if you keep on typing the same thing for a while. This is why my app includes a game mode that generates random - albeit unintelligible - sentences based on the 1000 most used Korean words.

Set your pace

Start today! Work through the lessons at your own pace. You'll notice progress before long ! After you've mastered the Korean keyboard layout, try out the Game Mode and increase your WPM rate.
