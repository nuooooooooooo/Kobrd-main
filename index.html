Typing on a korean keyboard, it's easier than you think.

Have you recently started to learn Korean but find yourself having to look up where a certain hangeul is on your computer keyboard every time you want write a sentence? Fret not! This is a problem every learner of the Korean language is bound to struggle with once they start wanting to type on a computer keyboard - myself included.

This is why I created this application, Name. With it, you'll start learning Korean dactylography from the beginning, the same way you did when you first learned how to type on a QWERTY keyboard. And you know what's more? You can learn it all for FREE!

It's free

Yes! You read it right, this application was made for free, because as a learner myself, I struggled for some time before getting the hang of it. And I think no one should have to struggle anymore. After all, it's 2020! This should have been done a long time ago.

It's gamified

While learning can be fun, it can sometimes be hard to keep motivated after you do the same thing for a while. This is why I included a leaderboard and several other features to make learning how to type fun, as it should always be.
